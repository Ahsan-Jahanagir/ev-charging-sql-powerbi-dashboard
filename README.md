# EV Charging Data Analysis (SQL + Power BI)

This project analyzes an **EV charging network dataset** using **SQL (SQLite)** for data exploration and **Power BI** for interactive visualization.

The goal of the project was to transform raw charging session data into meaningful insights about **revenue performance, charger usage, and station efficiency**.

---

## Project Workflow

1. **Data Storage**
   - The dataset is stored in a SQLite database.

2. **SQL Data Analysis**
   - Used SQL queries to explore and analyze charging data.
   - Combined tables using **JOIN** operations.
   - Calculated metrics using **SUM()**, **GROUP BY**, and **ORDER BY**.

3. **Power BI Visualization**
   - Built an interactive dashboard to visualize the insights.

---

## Key Analysis

The SQL analysis focuses on answering questions such as:

- Which **city generates the highest charging revenue**?
- Which **charger type (AC vs DC Fast)** produces more revenue?
- Which **charging stations perform best**?
